HireStart is a frontend web prototype for a platform that helps recruiters hire verified fresher candidates.
The project shows the UI and workflow of resume building, certificate verification, and a recruiter dashboard.

Features

Resume Builder UI

Certificate Verification Demo

Digital Resume Passport

Job Readiness Score (demo)

Recruiter Dashboard for browsing and shortlisting candidates

Tech Used

HTML, CSS, JavaScript

This project is a concept and UI prototype, and backend features will be added in future.
